# MIND: Memory Consistency & Action Control in World Models

> 论文: [MIND - Benchmarking Memory Consistency and Action Control in World Models](https://arxiv.org/abs/2602.08025)
> 机构: CSU-JPG (Colorado State University Joint Processing Group)
> 发表: 2026-02
> 代码: github.com/CSU-JPG/MIND

---

## 核心问题

World Model (WM) 在长程预测中是否保持记忆一致性？当视频预测帧数增加时，WM 是否能：
1. 维持物体/场景的视觉一致性？
2. 准确预测动作效果？
3. 保持空间推理的连贯性？

---

## 三大量化维度

### ℒ_lcm — Long-term Consistency (长程一致性)
衡量 WM 在长期视频预测中维持视觉一致性的能力。

**测试方法:**
- 输入: 初始帧序列 + 动作序列
- 输出: 未来 10/30/50/100 帧的预测
- 对比: 预测帧 vs 真实帧的 LPIPS 感知距离 + MSE

**关键发现:**
| 帧数 | MSE | 备注 |
|------|-----|------|
| 10帧 | 0.10 | 基线正常 |
| 30帧 | 0.18 | 开始衰减 |
| 50帧 | 0.26 | 显著下降 |
| 100帧 | 0.31 | 几乎不可信 |

**核心问题:** 当前 WM 在 >50 帧后一致性显著下降，主要表现为：
- 物体几何变形（杯子变歪、手臂缩短）
- 材质漂移（颜色/纹理不连续）
- 物理规律违反（物体悬浮、穿模）

---

### ℒ_gsc — Geometric Consistency (几何一致性)
衡量 WM 在动作执行后维持空间关系的能力。

**测试方法:**
- 轨迹追踪: 机器人执行"向前走10步→返回起点"
- 测量: 预测轨迹 vs 真实轨迹的终点偏差

**关键发现:**
- 向前→返回环路存在**累积误差**
- 单步误差 ~0.02m，10步后累积误差 ~0.15m
- 几何不稳定：WM 的非确定性生成导致误差累积

**核心问题:** 缺乏显式碰撞检测，角色-背景交互失效

---

### RPE — Relative Pose Error (相对位姿误差)
衡量 WM 预测动作效果的准确性。

**关键发现:**
- **有记忆反而更差**: 当测试动作空间与训练不一致时，有记忆上下文干扰推理
- 无记忆基线 vs 有记忆模型：后者在OOD动作场景下 RPE 高出 23%
- 说明当前 WM 的记忆机制是**条件依赖**而非**通用**

## 四大核心失效场景

### 1. 长程记忆漂移 (Long-horizon Drift)
```
问题: 上下文窗口有限，早期信息丢失
症状: >100帧预测几乎不可信
量化: LPIPS 从 0.05 (10帧) → 0.42 (100帧)
```

### 2. 动作-记忆干扰 (Action-Memory Interference)
```
问题: 记忆上下文在OOD动作场景下干扰推理
实验: 训练动作空间[0-90°]  测试[95-180°]
结果: 有记忆模型 RPE 比无记忆基线高 23%
```

### 3. 环路不一致 (Loop Inconsistency)
```
问题: 非确定性生成导致累积误差
实验: 闭环测试（预测→执行→预测）
结果: 开环预测 vs 闭环执行差异显著
```

### 4. 第三人称穿模 (3rd-Person Penetration)
```
问题: 缺乏显式碰撞检测
症状: 角色穿墙、手持物体穿过桌面
根因: 纯视觉生成无法保证几何一致性
```

## 研究机会

### 机会 1: 记忆压缩机制
```
研究方向: 选择性记忆 + 压缩表示
目标: 保持关键信息同时减少干扰
思路:
  - KL散度门控: 哪些记忆值得保留
  - 遗忘曲线建模: 模拟人类记忆衰减
  - 状态抽象: 高层状态 vs 低层像素
```

### 机会 2: 动作空间自适应
```
研究方向: 检测记忆干扰并动态解除
目标: OOD动作场景下自动关闭记忆条件
思路:
  - 不确定性估计: 当记忆置信度低时降权
  - 在线适应: 快速微调记忆模块
  - 混合架构: 有记忆/无记忆双路径
```

### 机会 3: 几何约束注入
```
研究方向: 将几何先验嵌入 WM
目标: 解决碰撞检测和空间推理问题
思路:
  - 深度图/占用网格作为额外输入
  - 物理引擎监督信号
  - 图神经网络建模物体关系
```

### 机会 4: 记忆-动作解耦训练
```
研究方向: 分别训练记忆模块和动作模块
目标: 消除干扰，提升泛化
思路:
  - 对比学习: 记忆无关的动作预测
  - 梯度阻断: 防止记忆干扰动作梯度
  - 分层预测: 抽象规划 + 精细控制
```
